## 💬 AI 질의 응답 기록

### 🔹 Q1. 이 코드는 파이썬 코드이다. 제공된 소스 코드를 분석해서 이 코드를 순서 되로 자세히 설명해줘
- **AI Answer**

1. pygame와 sys를 불러오고 pygame.init()으로 게임 준비를 시작한다.
2. set_mode((800, 600))으로 가로 800, 세로 600 크기의 창을 만든다.
3. WHITE와 BLUE는 흰색과 파란색을 저장한 색 변수이다.
4. while running: 반복문은 창이 켜져 있는 동안 계속 화면을 갱신한다.
5. 반복문 안에서 창 닫기 이벤트를 확인하고, 흰 배경 위에 가운데 파란 원을 그린다.
6. display.flip()으로 화면을 보여주고, 마지막에 pygame.quit()과 sys.exit()로 종료한다.
- **issue**
 처음 질문에 대한 답변이 너무 길어서 6줄로 요약해서 요청
- **Result**
  ✅ 성공

### 🔹 Q2. 방향 키보드 입력을 받아서 움직이게 수정해줘?
- **AI Answer**
 바뀐 핵심은 이거야.
x = 400, y = 300으로 원 위치를 변수로 저장
pygame.key.get_pressed()로 현재 눌린 키 확인

원을 (400, 300) 고정이 아니라 (x, y)에 그림

  ✅ 성공
